## About

PandaQuest is a fun, educational game where a panda must collect bamboo while avoiding enemies. This project is designed for AP Computer Science A students to learn and practice object-oriented programming concepts in an engaging way.

## Greenfoot Scenario

This repository includes a complete **Greenfoot-compatible scenario** that can be opened and run directly in Greenfoot.

2. Clone this repository
3. Open Greenfoot and go to `Scenario` ‚Üí `Open...`
4. Navigate to the `greenfoot-scenario` folder
5. Start playing and modifying the code!

## Learning Objectives

This project helps students learn:

- **Object-Oriented Programming**: Classes, objects, inheritance, encapsulation, polymorphism
- **Game Development**: Game loops (act method), collision detection, user input handling
- **Problem Solving**: Algorithm design, debugging, code organization
- **Software Design**: Class relationships, method design, documentation

## Game Overview

**Objective**: Control the panda to collect all bamboo while avoiding enemies

**Controls**: 
- Arrow Keys ‚Üí Move the panda

**Win Condition**: 
- Collect all 5 bamboo items

**Lose Condition**: 
- Get hit by enemies 3 times

**Features**:
- Health system with invincibility after taking damage
- Moving enemies with patrol AI
- Score tracking
- Wall obstacles and boundaries
